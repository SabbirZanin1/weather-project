# Weather Project

## About
This is a simple weather analysis project built with Python.  
The project uses monthly weather data to show temperature changes and different weather visualizations.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Features
- Temperature chart
- Animated graph
- Heatmap
- Pie chart
- Average temperature calculation
- Highest temperature day detection
